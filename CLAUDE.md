# depreciation_drill

簿記学習用ドリル集（静的HTML・PWA）。1ドリル1ファイルで、`index.html`（減価償却）ほか各 `*.html` が独立したページ。

## デプロイ（重要）

**このサイトは main ブランチから配信されている。作業ブランチにプッシュしただけではユーザーは変更を使えない。**

作業完了時は、作業ブランチへのプッシュに加えて、必ず main にもマージしてプッシュすること（ユーザーへの確認は不要）。

```
git fetch origin main
git checkout main && git merge <作業ブランチ> && git push origin main
```

## 変更時の注意

- 新しいドリルページを追加したら：
  - 全ページの `.nav` にリンクを追加する（`付随費用</a>` の後ろが末尾）
  - `service-worker.js` の `ASSETS` にファイルを追加し、`CACHE` のバージョン番号を上げる
- ナビやスタイルは既存ページ（例：`trade.html`）からコピーして統一する
- 問題生成は金額が割り切れる（整数になる）ように基数を設計し、node で大量生成テストをしてから出すこと
